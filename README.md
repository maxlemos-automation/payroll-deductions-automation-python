## üìç Project Overview
This project automates the **ETL (Extract, Transform, Load)** process for payroll deductions. It eliminates manual errors by converting raw HR Excel data into a strictly formatted TXT file compatible with the **Metiri ERP** (and adaptable to others like SAP/Bejerman).

It uses **Python (Pandas)** to handle large datasets efficiently, ensuring zero discrepancies in the final payroll import.

---

## ‚öôÔ∏è Key Features
- ** Automated Data Cleaning:** Instantly handles missing values (`NaN` ‚Üí `0`) and trims whitespace errors.
- ** Wide-to-Long Transformation:** Unpivots columns (Concept A, Concept B...) into rows required by database systems.
- ** Smart Mapping:** Translates human-readable names (e.g., "Gym Fee") into ERP system codes (e.g., `DED_045`) using an external dictionary.
- ** Metiri-Ready Export:** Generates the specific semicolon-separated (`.txt`) format required for bulk importing.
