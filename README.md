# Payroll Deductions Automation

This project automates the process of converting employee deduction data into a payroll-ready TXT file used by the **Metiri** payroll system.

It reads raw deduction data from Excel, maps deduction codes, formats the structure, and exports a clean .txt file ready for import.

---

### ‚öôÔ∏è What the automation does

- Reads a source Excel file with employee deductions (`employee_deductions.xlsx`)
- Cleans and reshapes data (missing values ‚Üí zero, wide ‚Üí long transformation)
- Maps deduction descriptions to system codes (`deduction_codes.xlsx`)
- Generates a payroll-import TXT file formatted for Metiri (semicolon `;` separated, no header)
